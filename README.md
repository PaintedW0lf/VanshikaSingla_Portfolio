# Vanshika Singla - Portfolio Website

Clean, modern, white-theme portfolio for a multi-disciplinary professional in Software Engineering, Data Engineering, Data Analytics, and Leadership.

## Structure

- `index.html` - Main landing page with all sections
- `project-detail.html` - Template for individual project pages
- `styles.css` - Main stylesheet with white theme
- `project-detail.css` - Styles for project detail pages
- `script.js` - Smooth animations and interactions

## Design Philosophy

- White background with soft greys
- Generous spacing and clean typography
- Subtle animations on scroll
- Executive-level tone
- Mobile responsive
